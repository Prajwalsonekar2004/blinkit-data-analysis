# Blinkit Data Analysis

This project explores Blinkit order, sales and delivery data using Python. It focuses on understanding sales trends, product performance, customer behavior and delivery patterns. The goal is to practice real-world data analysis skills and build a strong portfolio project for data analyst roles.

## Features
- Data cleaning and preprocessing
- Exploratory data analysis
- Time-based sales trends
- Product and category performance
- Delivery time insights
- Visualizations using Matplotlib and Seaborn

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## Project Structure
# blinkit-data-analysis
